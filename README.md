## Getting Started

To get a local copy up and running, follow these simple example steps.

### Prerequisites

* PHP8.3
  ```sh
  sudo apt update
  sudo apt upgrade
  sudo apt install -y php8.3
  ```

### Installation

2. Clone the repo
   ```sh
   git clone git@github.com:roma-mb/poo-php.git
   ```
